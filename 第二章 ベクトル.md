## 2.5 all()和 any()的使用
### 2.5.1 例子：寻找连续值
寻找由0，1组成的数列里，1的连续数列。
比方说在向量c(1,0,0,1,1,1,0,1,1)里，序列4开始连续序列长度为3，序列4，5，8，开始的话连续数列长度为2。  
编写一个函数``findruns()``,使得``findruns(c(1,0,0,1,1,1,0,1,1),2)``会返回数字(4,5,8)
```r
findrruns<-function(x,k){
  n<-length(x)
  runs<-NULL
  for(i in 1:(n-k+1)){
    if(all(x[i:(i+k-1)]==i))
      runs<-c(runs,i)
  }
  return(runs)
}
```
